THE FOLLOWING LIBRARIES ARE NEEDED TO SUCCESSFULLY RUN THIS PROGRAM

INSTALLATION (pip install --upgrade theano)
#KERAS
INSTALLATION (pip install --upgrade keras)
#TENSORFLOW
